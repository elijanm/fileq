plugin:list - List all installed plugins
plugin:publish - Package and publish a plugin to the internal registry
plugin:rollback - Fetch plugin from internal registry, verify, and install.
plugin:verify-all - Reverify all installed plugins (signature check)
plugin:new - Generate a new plugin folder template with standard structure and options
plugin:discover - Discover plugins and register them in the local database.
plugin:install - Fetch plugin from internal registry, verify, and install.
plugin:upgrade - Fetch plugin updated internal registry.
marketplace:sync - Sync available plugins from registry
migration:new - Create a new migration script for a plugin
migration:run - Run all pending migrations for a plugin.
migration:rollback - Rollback the most recent migration for a plugin.
